# Graph Client Package

In your composer.json add the following:
```
"trellis/domains-microservice": "dev-dev"
```
and then add this as well:
```
"repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:harris21/trellis-domains.git"
        }
    ],
```

and do a ```composer update```.
